Personal nix config for each of my machines.

## Install

### Clone repo

    git clone https://github.com/khaneliman/khanelinix.git ~/.config/khanelinix

### Automatic

```bash
cd ~/.config/khanelinix && sudo nixos-rebuild switch --flake . # linux
```
